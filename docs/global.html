find(collection, condition) → {Promise}

The find method returns a value in the collection, if an element in the collection satisfies the provided testing function. Otherwise undefined is returned. Similar to Array.find. REF: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/find
Parameters:
Name Type Description
collection Array | Object Collection to search.
condition function Condition test function. Should return true if match, otherwise false.
